# Die Kryptographiewende

ist schon da. Die Kryptographie zur sicheren Kommunikation wendet sich von pre-quanten-sicheren (klassischen), über hybride, zu post-quanten-sicheren Algorithmen.

Die Kryptographie sind jene Algorithmen, welche die Kommunikation sicher und vertrauenswürdig machen, indem sie wesentliche Schutzziele der Informationssicherheit verwirklichen.

Das Gelingen dieser Wende liegt an uns.
